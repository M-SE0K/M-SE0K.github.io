# 챌린지

개인적으로 진행하는 다양한 도전과 학습 과정을 기록합니다.

## 진행 중인 챌린지

### 1일 1알고리즘
매일 하나씩 알고리즘 문제를 풀며 코딩 실력을 향상시키는 도전입니다.

**진행 기간**: 2024년 1월 ~ 현재  
**플랫폼**: 백준, 프로그래머스, LeetCode  
**목표**: 365일 연속 문제 해결

### 대회 및 코딩 테스트
다양한 프로그래밍 대회와 코딩 테스트에 참여하여 실전 경험을 쌓습니다.

**참여 대회**:
- 삼성 SW 역량테스트
- 네이버 부스트캠프 코딩테스트
- 각종 해커톤 및 알고리즘 대회

### 100일 코딩 챌린지
100일 동안 매일 코딩을 하며 꾸준한 학습 습관을 만드는 도전입니다.

**진행 방식**:
- 매일 최소 1시간 이상 코딩
- GitHub에 일일 커밋 기록
- 학습 내용 블로그 포스팅

## 완료된 챌린지

### React 마스터하기
React와 TypeScript를 활용한 프로젝트 개발에 집중한 3개월 챌린지

**기간**: 2024년 6월 ~ 2024년 8월  
**성과**: 5개의 완성된 프로젝트, 포트폴리오 사이트 구축

### 웹 접근성(A11y) 학습
웹 접근성 가이드라인을 학습하고 실제 프로젝트에 적용한 챌린지

**기간**: 2024년 9월 ~ 2024년 10월  
**성과**: WCAG 2.1 가이드라인 숙지, 접근성 검사 도구 활용법 습득
